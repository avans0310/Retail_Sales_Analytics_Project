# Retail Sales Analytics Dashboard

## Project Overview
This project analyzes retail sales data using Python and Power BI to identify sales trends, category performance, and customer insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI
- SQL

## Features
- Total Sales KPI
- Average Sales KPI
- Sales by Product Category
- Sales by Gender
- Monthly Sales Trend
- Interactive Filters (Gender, Product Category, Date)

## Project Structure
- Dataset for sales analysis
- Python data analysis script
- Power BI dashboard
- Charts and outputs

## Key Insights
- Product category sales comparison
- Gender-based sales distribution
- Monthly sales trends analysis
- Interactive business reporting dashboard
